# Aurora Network CH3000 - DX3515

The DX3515 is a digital transponder connected to the Aurora Networks CH3000 chassis.

## About

This connector is automatically generated by the parent connector [Aurora Network CH3000](xref:Connector_help_Aurora_Network_CH3000), from version 1.0.0.1 onwards.

## Installation and configuration

### Creation

This connector is used by DVE child elements that are **automatically created** by the parent connector [Aurora Network CH3000](xref:Connector_help_Aurora_Network_CH3000).
